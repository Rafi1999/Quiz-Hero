1. What is the difference between map, filter, foreach, find?
Ans: map() executes the same code on every element in an array and returns a new array with the updated elements. filter() checks every element in an array to see if it meets a certain criteria and returns a new array with the elements that return truthy for the criteria. forEach(), is used to execute the same code on every element in an array but does not change the array and it returns undefined.find() is used to select a single element from an array.
